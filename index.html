SQLite3

  • Create a database named "test.db" with the SQLite3 cli. Run the command "sqlite3 <database name>.db" in the terminal.
  • Open test.db with the cli
    1. Start the cli by running "sqlite3" in the terminal.
    2. Open the database with ".open FILENAME"
  • Create a table called "users" with columns "id (primary key) and name (text) yearbook_quote (text)". Run the command "CREATE TABLE users(id INTEGER PRIMARY KEY, name TEXT);"
  • Insert at least 3 new users. Run the command "INSERT INTO users(name, yearbook_quote) VALUES('<name>', '<yearbook_quote>');"
  • Select all users. SELECT * FROM users;
  • Select all users names only. SELECT names and yearbook_quotes FROM users;
  • Change the name of a user. UPDATE users SET name='<name>' WHERE id=<id>;
